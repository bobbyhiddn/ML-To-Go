# Week 4: Neural Networks for Text

## Learning Objectives
- Build neural network classifiers with PyTorch/TensorFlow
- Understand feedforward, CNN, and RNN architectures
- Implement text classification with deep learning
- Learn about training loops, loss functions, and optimizers

## Topics Covered
- Feedforward neural networks for text
- Convolutional Neural Networks (CNNs) for text
- Recurrent Neural Networks (RNNs, LSTMs)
- Training, validation, and testing workflows

## Exercises
1. Build a simple feedforward classifier
2. Implement a CNN text classifier
3. Create an LSTM-based classifier
4. Compare deep learning vs. classical ML performance
