# Week 4: Text Extraction Pipeline

## Overview
Extract text from any document format: PDFs, Word, Excel, PowerPoint, and scanned images.

## Learning Objectives
- Extract text from PDFs reliably
- Process Office documents (DOCX, XLSX, PPTX)
- Handle OCR for scanned documents
- Build unified extraction interface

## Notebooks
1. `01_pdf_extraction.ipynb`
2. `02_office_documents.ipynb`
3. `03_images_ocr.ipynb`
4. `04_unified_extractor.ipynb`

## Key Components
- `extractors/pdf_extractor.py`
- `extractors/docx_extractor.py`
- `extractors/xlsx_extractor.py`
- `extractors/unified.py`

## Deliverable
Process 1000+ mixed documents through unified extractor.
