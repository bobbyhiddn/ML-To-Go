# Week 1: Document Embeddings

## Learning Objectives
- Understand text embeddings and vector representations
- Learn about word2vec, GloVe, and modern sentence embeddings
- Implement document embedding using sentence-transformers
- Explore embedding similarity and distance metrics

## Topics Covered
- Text preprocessing and tokenization
- Word embeddings vs. sentence embeddings
- Cosine similarity and semantic search
- Dimensionality reduction (PCA, t-SNE)

## Exercises
1. Load and preprocess sample documents
2. Generate embeddings using sentence-transformers
3. Calculate similarity between documents
4. Visualize embeddings in 2D space
