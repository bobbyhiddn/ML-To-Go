# Week 2: Feature Extraction

## Learning Objectives
- Extract meaningful features from documents
- Learn TF-IDF and bag-of-words representations
- Understand n-grams and feature engineering
- Process different document formats (PDF, DOCX)

## Topics Covered
- Document parsing (PDF, DOCX)
- TF-IDF vectorization
- N-gram features
- Feature selection and dimensionality reduction

## Exercises
1. Parse PDF and DOCX documents
2. Implement TF-IDF feature extraction
3. Compare different feature extraction methods
4. Create a feature engineering pipeline

- scikit-learn TfidfVectorizer
