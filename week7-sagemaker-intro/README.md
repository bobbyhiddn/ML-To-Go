# Week 7: Introduction to AWS SageMaker

## Learning Objectives
- Set up AWS SageMaker environment
- Understand SageMaker notebooks and training jobs
- Prepare data for cloud training
- Run training jobs on SageMaker

## Topics Covered
- AWS SageMaker overview and setup
- SageMaker notebooks and instances
- Data preparation and S3 storage
- Training jobs and built-in algorithms
- SageMaker Python SDK

## Exercises
1. Set up AWS account and SageMaker
2. Upload training data to S3
3. Run a training job on SageMaker
4. Monitor training with CloudWatch
